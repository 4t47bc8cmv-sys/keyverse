    First Steps on Mac

    Volume 1, Game 1 · Keyverse — Mac Shortcut Trainer

    First Steps on Mac is the starting point of the Keyverse series. It drills the everyday macOS keyboard shortcuts that make the biggest difference the fastest — opening apps with Spotlight, copying and pasting, switching and quitting apps, saving and finding text, and capturing the screen. If you are new to the Mac, or you have always reached for the menus, this is where the muscle memory begins.

    What this game teaches

    • Level 1: Desktop Basics — Spotlight, Undo, Go Back, Files
    • Level 2: Clipboard & Data — Copy, Paste, Cut, Select All
    • Level 3: App Control — Switch, Quit, Minimize, Force Quit
    • Level 4: Docs & Web — Save, Find, Refresh, Bold
    • Level 5: System Mastery — Screenshots & Lock Screen

    How to play

    1. Pick a level from the menu to see the shortcuts it covers.
    2. Read the study list, then press Start Practice.
    3. When prompted, perform each shortcut: press the real keys on your keyboard, or tap the on-screen modifier and action keys, then press Return.
    4. Repeat mode shows the answer first as training wheels; Test Mode hides it so you can check your recall.
    5. Complete every shortcut in a level, then choose Next Level to continue to the next set.

    The game is free, runs entirely in your browser with no download or account, works on desktop, laptop, tablet, and phone, and is available in 12 languages. This reference below stays in English.
